# Building Permit Application - Employee Flow

## Login as HOD

https://djibouti.tekdinext.com/digit-studio/employee/login

## View Building Permit Applications

https://djibouti.tekdinext.com/public-service/v1/search?tenantId=dj

## View Application Details

https://djibouti.tekdinext.com/public-service/v1/application/SVC-DJ-BPA-BPA_PCO-02/{applicationNumber}

## Get workflow details

https://djibouti.tekdinext.com/egov-workflow-v2/egov-wf/businessservice/_search?tenantId=dj&businessServices=BPA_PCO

## Assign to Agent

```bash
curl 'https://djibouti.tekdinext.com/public-service/v1/application/SVC-DJ-BPA-BPA_PCO-02' \
-X 'PUT' \
-H 'accept: application/json, text/plain, */*' \
-H 'accept-language: en-GB,en-US;q=0.9,en;q=0.8' \
-H 'content-type: application/json;charset=UTF-8' \
-H 'x-tenant-id: dj' \
--data-raw '{
  "Application": {
    "id": "{applicationId}",
    "applicationNumber": "{applicationNumber}",
    "tenantId": "dj",
    "workflow": {
      "action": "ASSIGN_TO_AGENT",
      "comment": "Assigning to agent for review",
      "assignees": [
        {
          "uuid": "{agentUuid}",
          "roles": [
            {
              "code": "BPA_AGENT",
              "tenantId": "dj"
            }
          ]
        }
      ],
      "businessService": "BPA_PCO"
    }
  },
  "RequestInfo": {
    "apiId": "Rainmaker",
    "authToken": "{authToken}",
    "userInfo": {
      // HOD user info
    }
  }
}'
```

-H 'accept: application/json, text/plain, */*' \

  },

  }
}'
